
Проєкт: Бульйонний маніяк — оновлений стиль (гравюра/комікс) з паралаксом і сакурою
Інструкції:
1. Розпакуй архів і відкрий index.html у браузері.
2. SVG-шари в папці /svg; швидкість паралакса регулюється у js/background.js
3. Сакура — canvas overlay (js/background.js).
4. Меню читається з products.json; кошик — в localStorage (bm_cart).
